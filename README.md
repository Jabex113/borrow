# Mortgage Calculators

 React + TypeScript 

- **Loan Savings**: credit score → APR → monthly payment → total interest
- **How Much Can I Borrow**: income ratios (29–30% housing, 36% DTI) → max loan amount

## Getting Started

Install dependencies and start the dev server.

```
npm install
npm run dev
```

## Build

```
npm run build
```
